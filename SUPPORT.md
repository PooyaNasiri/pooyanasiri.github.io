# Support

## Need Help?

If you are forking this portfolio to use as your own template and run into issues, here is how you can get help:

1.  **Check the README**: Make sure you have followed the setup instructions, particularly regarding the **GitHub Stats** Vercel setup.
2.  **Search Issues**: Someone might have already asked your question.
3.  **Open an Issue**: If you found a bug in the code, please [open a GitHub issue](https://github.com/PooyaNasiri/pooyanasiri.github.io/issues/new).
